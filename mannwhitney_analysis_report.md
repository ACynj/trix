## 3. 总结

### 显著性水平说明
- ***: p < 0.001 (高度显著)
- **: p < 0.01 (高度显著)
- *: p < 0.05 (显著)
- 无标记: p >= 0.05 (不显著)

### 效应量说明 (Cohen's d)
- |d| < 0.2: 微小效应
- 0.2 ≤ |d| < 0.5: 小效应
- 0.5 ≤ |d| < 0.8: 中等效应
- |d| ≥ 0.8: 大效应

### 注意事项
1. Mann-Whitney U Test 适用于独立样本的比较。
2. 对于配对数据，Wilcoxon signed-rank test 可能更合适。
3. p值小于0.05通常被认为是统计显著的。
4. 效应量用于评估差异的实际意义，而不仅仅是统计显著性。
