## 3. 总结

### 显著性水平说明
- ***: p < 0.001 (高度显著)
- **: p < 0.01 (高度显著)
- *: p < 0.05 (显著)
- 无标记: p >= 0.05 (不显著)

### 效应量说明 (Cohen's d)
- |d| < 0.2: 微小效应
- 0.2 ≤ |d| < 0.5: 小效应
- 0.5 ≤ |d| < 0.8: 中等效应
- |d| ≥ 0.8: 大效应

### Wilcoxon Signed-Rank Test 说明
1. **适用场景**: 配对数据（同一数据集上的两个结果）
2. **优势**: 比 Mann-Whitney U Test 更适合配对数据，统计功效更高
3. **假设**: 差异的分布是对称的
4. **解释**: p值小于0.05表示两个方法之间存在显著差异
5. **效应量**: Cohen's d 用于评估差异的实际意义
